# Gestor de tareas
  Aplicación que permite crear, borrar o marcar como completadas tareas así como visualizarlas según su estado.

  Tecnologías y herramientas utilizadas:
  
    1. JavaScript
    2. HTML
    3. CSS
    4. Vite
    5. VSCode
    6. uuid
